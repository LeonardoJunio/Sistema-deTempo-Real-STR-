### :gear: Settings:

* Launch dockerfile:
```bash
docker build -f c_cpp.Dockerfile -t my-gcc-app ./ && docker run -it --rm -v ./:/App my-gcc-app
```

* Run the following commands:
```bash
cd threads-trains/
make
./bin/app 
```
